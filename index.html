img-txt-hybrid.js demo:

Thank you for checking out this jQuery plugin demo! There are several things you can do while you're here.

  • Copy-paste images from other websites into the big text-box OR
  • Drag-and drop images from your computer.
  • Type out your life story!
    • Basic formatting is enabled:
    • CTRL + b: bold
    • CTRL + i: italics
    • CTRL + u: underline
    • Only formatting tags get through to the backend unescaped. (Although sanitizing inputs is ALWAYS recommended).
  • Click either of the buttons below the text area to see what the built-in functions can send to the backend.
  • Fork this repository for your own nefarious purposes.
  • Download the Javascript file to use on your own dastardly website (includes minified JS file)!
